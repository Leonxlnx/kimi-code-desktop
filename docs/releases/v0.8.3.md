# Kimi Code Desktop 0.8.3

This is the first clean public-source release of Kimi Code Desktop, an unofficial Windows desktop client and agent harness for Kimi Code CLI.

## Highlights

- Sign in with any supported Kimi account and use that account's plan, quota, models, and sessions.
- Organize multiple resumable coding chats under local project folders.
- Keep standalone chats separate from project files, Git state, terminal, and preview tools.
- Queue, steer, stop, copy, and revert turns through a compact native composer.
- Use Kimi commands, skills, plugins, MCP servers, and official subagents from the desktop interface.
- Review Git changes, stage and unstage files, inspect diffs, and create commits.
- Run workspace commands in the integrated terminal.
- Let Kimi open, resize, reload, and capture an isolated localhost app preview.
- Customize theme, typography, density, shortcuts, layout, and panel placement.
- Install cryptographically signed application updates from inside the app.

## Security and privacy

- Kimi authentication remains owned by the official CLI.
- The app never reads or copies Kimi credential contents.
- Packaged renderer connections use a random per-launch token.
- Workspace files are path-validated, text-only, and size-limited.
- Preview accepts only `localhost` and `127.0.0.1`.
- Preview screenshots use a temporary, isolated Microsoft Edge profile.
- No app telemetry is implemented.

## Install

Download `Kimi-Code-Desktop-0.8.3-x64-setup.exe` below and run it on Windows 10 or Windows 11.

The release also includes:

- `Kimi-Code-Desktop-0.8.3-x64-setup.exe.sig` for Tauri updater verification
- `latest.json` for the signed update feed
- `SHA256SUMS.txt` for installer checksum verification

Windows SmartScreen may show an unrecognized-publisher warning because Microsoft Authenticode signing is not configured yet. Tauri updater signatures remain enabled and are separate from Authenticode.
